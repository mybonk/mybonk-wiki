# NixOS Platform Management Workshops

Understanding how a NixOS system is managed through practical examples of declarative systems configuration, reproducible deployments, container orchestration and more.

## Workshops

| Workshop | Description |
|----------|-------------|
| [Workshop 1](./workshop-1/) | **Run NixOS in a VM** - Set up your first NixOS virtual machine |
| [Workshop 2](./workshop-2/) | **Configure Services Declaratively** - Deploy services using NixOS configuration |
| [Workshop 3](./workshop-3/) | **Override Packages** - Customize software using Nix package overrides |
| [Workshop 4](./workshop-4/) | **Deploy Integrated Stacks** - Use pre-built NixOS modules for complex services |
| [Workshop 5](./workshop-5/) | **Build Custom ISOs** - Create pre-configured installation media with SSH |
| [Workshop 6](./workshop-6/) | **Manage Containers via CLI** - Create and manage containers independently |
| [Workshop 8](./workshop-8/) | **Container Networking** - DHCP, DNS, and NAT for container infrastructure |
| [Workshop 9](./workshop-9/) | **Service Orchestration in Containers** - Deploy complex services across containers |

## Getting Started

Start with [Workshop 1](./workshop-1/) to set up your NixOS environment, then progress through the workshops in order. Each workshop builds on concepts from previous ones.

## Prerequisites

- Basic Linux command-line knowledge
- A machine capable of running virtual machines (8GB+ RAM recommended)
- If you're not in the mood, pick a book from our list in [baby-rabbit-holes#books](./baby-rabbit-holes.md#books) and come back when you're ready
